• М. Горький читав з швидкістю чотири тисячі слів в хвилину.
  • Т. Едісон читав відразу 2–3 рядки, запам'ятовуючи текст мало не сторінками завдяки максимальному зосередженню.
  • При швидкому читанні стомлюваність очей менша, ніж при повільному.
  • 95% людей читають дуже повільно — 180–220 слів в хвилину (1 сторінку за 1,5–2 хвилини).
  • Рівень розуміння при традиційному читанні складає 60%, при швидкому — 80%.
  • Бальзак прочитував роман в двісті сторінок за півгодини.
  • Очі людини з середніми навиками читання роблять на одному книжковому рядку 12–16 зупинок, що читає швидко 2–4 зупинки.
