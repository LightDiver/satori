  • Мы принимаем статьи от авторов или правообладателей следующих языках: украинский, английский и русский.
    •             
  • Публикация статей проводится зарегистрированными пользователями.
    •

    •

    •             
  • Мы не проводим выплату гонорара авторам материалов, не взимаем плату за место для размещения статьи.

  • Статья для общего просмотра появится после акцепта редактора.
    •
