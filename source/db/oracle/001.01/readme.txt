Назначение файлов:

00-base.sql - базовые скрипты (зарезервировано)
01-user.sql - дополнительные скрипты касательно пользователя (зарезервировано)
02-main.sql - скрипты создания таблиц, первичных и внешних ключей, индексов, типов, последовательностей
03-exts.sql - скрипты создания хранимых процедур и функций на иных языках                              
04-spec.sql - скрипты создания спецификаций пакетов
05-head.sql - скрипты шаблонов процедур и функций
06-body.sql - скрипты создания (сборки) тел пакетов из папки "06-body"
07-proc.sql - скрипты создания процедур
08-func.sql - скрипты создания функций 
09-trig.sql - скрипты создания триггеров

11-load_table.sql - таблиця для імпорту LOB об'єктів sql*loader

20-init.sql - скрипты инициализации (базовые) для сборки релиза           

98-updt.sql - скрипт запуска всех предыдущих скриптов (полная сборка БД)
