Структура каталога и назначение командных файлов:

001.01 - скрипты для сборки базовой версии БД

create_data_base.sql - создание базы данных
create_work_user.sql - создание пользователя


Некоторое количество простых правил:
 
1. Приветствуется стремление к освоениию и применению даосской практики символического освоения пространства
   (Фэн-шуй). Помним, что код чаще ЧИТАЕМ чем ПИШЕМ и чаще ЧУЖОЙ чем СВОЙ. Тобто - "Цин нан ао юй", япона мать.

2. Весь sql-код распределяется по файлам (согласно принадлежности объекта) 

3. Файл может состоять из разделов, логически объединяющих разные объекты в одном файле
   Именование раздела:
   /* -------------------------------- */
   /*        {LOGICAL_OBJECTS}         */
   /* -------------------------------- */

   Например, имя для раздела "таблицы" в 02-main.sql 
   /* -------------------------------- */
   /*             TABLES               */
   /* -------------------------------- */
